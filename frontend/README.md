# HepaRadar Frontend

Врачебный дашборд для поиска пациентов с риском болезней печени и ХВГ.

## Стек
- React, Vite, TypeScript
- Tailwind CSS
- shadcn/ui (базовые компоненты)
- Tremor (графики и KPI-карточки)

## Запуск
1. Установите зависимости: `npm install`
2. Запустите dev-сервер: `npm run dev`

## Окружение (Mock vs API)
Приложение поддерживает переключение между локальными моками и реальным бэкендом без изменения кода компонентов.
В файле `.env` укажите:
```env
# Включить моки (true) или реальное API (false)
VITE_USE_MOCKS=true

# URL реального бэкенда
VITE_API_BASE_URL=http://localhost:8080
```

## Демо-логины
- Логин: `doctor` (или `coordinator`, `admin`) - имеет права формировать направление.
- Логин: `viewer` - права только на просмотр, без кнопки направления.
Пароль может быть любым (от 4 символов).
